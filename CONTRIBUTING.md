# Contributing to facial-recognition-databases

Thanks for helping keep this list accurate and useful! This repo is maintained by [FacePrivacy.ai](https://faceprivacy.ai) and accepts contributions from anyone.

## Ways to contribute

- **Add a service** that isn't on the list yet.
- **Update an entry** (link changed, company renamed, opt-out process changed, etc.).
- **Remove an entry** that is defunct or otherwise no longer relevant.
- **Fix typos / formatting** in the README.
- **Suggest a new section** if you think the categorization can be improved.

## How to submit a change

### Option 1 — Pull Request (preferred)

1. Fork this repository.
2. Create a branch: `git checkout -b add-service-xyz`.
3. Edit `README.md` and add or update the entry in the appropriate table. Keep the existing format:

   ```
   | Name | https://link | Short neutral description |
   ```

4. Commit your change with a clear message, e.g. `Add XYZ to enterprise vendors`.
5. Open a Pull Request describing what you changed and (where possible) include a source for the information.

### Option 2 — Issue

If you don't want to edit the file directly, open an [Issue](../../issues/new) with:

- The service name and website
- Which section it should go in
- A short neutral description
- Any opt-out / removal info you know about
- A source / link if available

## Style guidelines

- **Be neutral and factual.** No marketing language, no personal opinions, no accusations.
- **Use the official website** for the link when possible.
- **Keep the description short** — one sentence is ideal.
- **Mention opt-out / removal** briefly if the operator provides a process.
- **Alphabetize within sections** when it makes sense; otherwise keep similar services grouped.
- **One service per row.**

## What we won't accept

- Personal data, individual case files, or anything that identifies a private person.
- Instructions for misusing facial recognition tools (e.g., scraping, identifying strangers, bypassing consent).
- Promotional or affiliate links.
- Duplicates of existing entries (update the existing row instead).
- Services that don't actually do facial recognition or face search.

## Code of conduct

Be respectful. Disagreements about whether to include a service are fine — personal attacks are not. Maintainers reserve the right to close PRs or issues that violate this.

## Questions?

Open an [Issue](../../issues/new) or reach out via [FacePrivacy.ai](https://faceprivacy.ai).
