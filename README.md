# Facial Recognition Databases & Services

A community-maintained list of facial recognition services, search engines, and databases. This repository is maintained by [FacePrivacy.ai](https://faceprivacy.ai) and is open to contributions from anyone.

If you know of a service that should be added, removed, or updated, please open a Pull Request or an Issue. See [Contributing](#contributing) below.

> Disclaimer: This list is provided for informational and privacy-research purposes only. Inclusion does not imply endorsement. Information may change — always verify the current opt-out / removal process directly with the operator.

---

## Consumer / Public Face Search Engines

Services that index public photos and allow anyone to search for a face.

| Service | Website | Notes |
|---|---|---|
| PimEyes | https://pimeyes.com | Public face search engine; opt-out form available. |
| FaceCheck.id | https://facecheck.id | Face search indexing public photos; free removal requests. |
| EyeMatch.ai | https://eyematch.ai | AI facial recognition search platform; opt-out form. |
| ClarityCheck | https://claritycheck.com | Facial recognition search platform with opt-out verification. |
| Lenso.ai | https://lenso.ai | AI reverse image search with face matching; permanent opt-out. |
| ProFaceFinder | https://profacefinder.com | Face recognition and reverse image search; opt-out form. |
| Precheck.ai | https://precheck.ai | AI-powered face search platform; data deletion on request. |
| FindClone | https://findclone.ru | Russian face search engine indexing VKontakte photos. |
| Search4Faces | https://search4faces.com | Russian face search indexing VK, Odnoklassniki, TikTok, ClubHouse. |
| Social Catfish | https://socialcatfish.com | Reverse image search and people search platform; opt-out available. |

## Contributing

We welcome contributions! To add or update a service:

1. Fork this repository.
2. Edit `README.md` and add the service to the appropriate table, keeping the existing format: `| Name | https://link | Short neutral description |`.
3. Open a Pull Request describing what you added or changed and (where possible) a source for the information.

You can also open an [Issue](../../issues) to suggest a change without editing the file directly.

### Guidelines

- Keep descriptions short, neutral, and factual.
- Prefer the company's official website for the link.
- If a service offers an opt-out / removal process, mention it briefly.
- Do not include personal data, individual case files, or instructions for misuse.
